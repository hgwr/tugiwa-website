Tugiwa

iOS application for personal task management.

Personal Task Management with Tugiwa

  • With Tugiwa you can display reminders and calendar data in a unified way.
  • You can set granularity and deadline for reminder tasks.
  • Tugiwa recommends the next task to be taken in considering the deadline and granularity set for each task.

Recommended for

  • those who have multiple projects or tasks and have difficulty in making prospects as to whether you can accept new tasks
  • those who have periodic meetings and tasks around the week and need to perform tasks with deadlines by sewing between them

Features of Tugiwa app

  • Tugiwa will automatically organize the tasks, so you can pick up the next task without thinking about anything.
  • You can find free time between appointments.
  • With Tugiwa you can visually display the disposable time of each week, the availability of the schedule, the degree of busyness, so it helps you to decide whether you can enter a new task.
  • The data referred to by Tugiwa app is those of Calendar app and Reminder app, so you can use existing data as it is.
  • Tasks registered with Tugiwa app are saved as a reminder task, so if you have multiple iOS devices (iPhone + iPad etc.) you can share data between them via iCloud.

How to use Tugiwa app

  • Enter events and tasks in iOS Calendar app. If you have already entered, Tugiwa app will refer to it as it is.
  • Add a new list for Tugiwa with Reminder app. Our recommendation is to make a list for each your project.
  • On the Preference tab of Tugiwa app, select the list of reminders to use.
  • Enter a task with Tugiwa app. Alternatively, edit the task you entered in the list of existing reminders on Tugiwa app.
  • At that time, enter the task deadline and granularity (how long it will take).
  • On the Tugiwa tab of Tugiwa app, check the task to be next.

Tugiwa Tab の使用

  • Tugiwa タブでは次の取り掛かるべきタスクの候補を見ることができます。
  • タスクをリコメンド順か締め切り順で表示できます。

タスク Tab

  • このタブではタスクの新規追加、削除、検索、完了済みのタスクの表示などができます。
  • リマインダー app と違い、タスクの編集をするときに締め切りや粒度などのデータを追加できます。

スケジュール Tab

  • 日毎の予定を確認できます。

カレンダー Tab

  • 日ごとの忙しさ具合を確認できます。
  • タスクの締め切りと粒度を勘案し、自動的にタスクをスケジュールに当てはめていった結果を確認できます。

カレンダー Tab の週表示

  • カレンダー Tab で週を選択すると、その週の日ごとの忙しさ具合の詳細を確認できます。

設定 Tab

  • Tugiwa app で使用するカレンダー、リマインダーのリスト、休日用のカレンダーを設定できます。
  • 作業可能な曜日(平日の定義)や、毎日の作業開始と終了時刻を設定でき、その時間の間にタスクを割り当てるようにできます。
  • タスクのリコメンド順を自分で調整できます。
  • たとえば、カレンダーのスケジュールタイトルとマッチするタスクを優先したり、
  • 期限は特にないけどなるべく早く終わらせたいタスクを優先させたり、
  • 締め切りを過ぎたタスクを優先させたり、逆に締め切りが過ぎていない間に合いそうなタスクを優先させたりできます。

Privacy Policy

iOS Tugiwa app を使うと、リマインダーやカレンダーのデータを統合して取り扱い、個人のタスク管理を便利に行うことができます。 お客様が Tugiwa app を安心してご利用いただけるよう、以下では Tugiwa app のプライバシーポリシーについて説明します。

Tugiwa app は、 iOS 通してリマインダーとカレンダーのデータにアクセスします。 そのため、Tugiwa app 初回起動時にリマインダーとカレンダーのデータへのアクセスの許可を求めます。 このアクセス許可については、 iOS の設定アプリ内でいつでも変更できます。

Tugiwa app は、リマインダーとカレンダーのデータを Tugiwa app の中でのみ利用します。 前述の通り、 Tugiwa app は iOS を通してリマインダーとカレンダーのデータにアクセスします。 具体的には、Tugiwa app は Apple の提供している EventKit API を使用しています。 結果として、Tugiwa app はお客様がお使いの iCloud や外部のカレンダーサービスにアクセスすることになります。 しかし Tugiwa app は以下の様なことはしません。 たとえば Tugiwa app は、お客様がご利用していない外部サイトにお客様のデータを送信したり、お客様のデータを収集して利用することはありません。
